JavaScript != JAVA


- Ahhhhh... your IPA on tap please. 24oz.
- Well while we wait for the 3rd wheel do you want to hear something that will blow your mind?

JavaScript it is a really popular programming language!
Let me tell you why.

First JAVA is not JavaScript. They are different breads.

Java is a Compiled programming language and JavaScript is a, guess... yes a Scripting one.
The difference between Compiled and Scripting is huge.

When you use a compiling language your code has to be compiled by a special program call compiler.
A compiler changes your code into a language that machine can understand. With a compiler language you dont run the code that you make but the result of the compiled process, which is a new file. A new file delivered from the compiler. Confused???

On the other hand, with Scripting language you dont need a compiler but you need an interpreter. Every browser is an interpreter and they can easily read line-by-line code that you write in JavaScript and work with it.

With a compiler you need to run separate compile process for Windows, Mac or Linux, but you can run JavaScript on any browser and it does not matter in which OS.

My friend dont you think I could be the most popular guy if I speak one language that girls of any culture can understand??

And in top of that, you can do all this kind of things using JavaScript, which by the way, is an Object Oriented programming language easy to learn and light to run.

But in order be or go happy, I think I have to create the conditions to that. Creating those conditions makes me feel proud and by the pillars that sustain my pride I can go to sleep satisfied. I can remember moments of my life that all those factors came together and collide in one event or day, like my wedding. Or like the day I knew my wife was pregnant or the first time I heard my babys heart beating.

This are some of values that help me create conditions to go happy in life:

Wisdom
Responsibility
Close Relationships
Community
Creativity
Nature

All this because there was a point back there in time, that I consciously decided to load my back pack with that RESPONSIBILITY and it will be always available for me.

One thing is to say, and other one is to do. I try but actually I live those values in a scale 1-5 like this:

Wisdom 3

Because there are some values that is hard to work on or to actually achieve, plus 5 is perfection and I dont have to be perfect :)

Comparing Web sites


Vimeo

Is one of my favorites websites because it let me discover shorts and documentaries that I could never find other way. It is easy to use and they have staff picking good material and artists for you. Their recommendations are fantastic and usually takes your to some unexplored corner of reality or fantasy.
I love the elegant way that the videos Meta data is displayed and how people really interact around it.
Vimeo makes my day several times a week.
Compare to Youtube I find Vimeo more relaxing to use, less add, less consumer hunting oriented and really whiling to help artists to improve or people to start doing good art.

Liftinteractive

These guys came up with an awesome way to use CSS. They are an agency and they are trying to educate their clients on what they do.
They make it really simple, elegant and pretty. I feel really engage with this guys vision and mission just by putting my eyes on their index. A full-page picture showing the culture, relax environment and professionalism.
They use the word LEARN. Learn more about us. Study us. Hire us.
I like their simplicity and the way they display their information. Their colors are not dark at all and matches with their life style or a least with my interpretation of it.

Sedona Hot Yoga

I use this website every week to check on schedule, classes, teachers and if there are any cancelation. Thats why is one of my favorites; they update it every week at least two times. And they have a Mobil version!
But their colors scheme is terrible. It stress me out, I dont like vising this website but I have to. I think it has basic design and it doesnt engage. For me it doesnt communicate their values as a yoga studio.
The way they display information is has a huge lack of elegancy.
